# 在线表格填报工具 - 前端

## 项目简介
本项目是在线表格填报工具的前端部分，旨在提供一个直观、高效的界面，用于表格切分、任务管理和数据填报。用户可以通过该界面上传表格、设置填报条件、分发任务，并跟踪填报进度。

## 项目功能

### 表格管理
- 支持多种格式表格文件上传（.xls/.xlsx/.csv/.et）
- 智能解析表格内容，自动提取表头和数据
- 提供表格预览功能，方便用户确认表格内容
- 支持根据指定列自动拆分表格，生成子表格

### 任务管理
- 创建和配置填报任务
- 设置表格数据验证规则，确保数据质量
- 生成可共享的填报链接
- 实时跟踪各子任务的填报状态和进度

### 数据填报
- 提供类似Excel的在线编辑体验
- 实时数据验证，防止填报错误
- 支持数据暂存和反复修改
- 提供清晰的操作指引和反馈

## 应用结构说明

### 核心页面说明

- **Home.vue**：首页，提供表格上传入口，显示历史任务列表，支持根据任务ID快速跳转。用户可以在此页面查看所有已创建的任务，并选择加载新的表格开始任务。

- **TaskGeneration.vue**：任务生成页面，提供表格拆分功能和结果预览。用户可以选择要拆分的列，系统会自动根据选定列的值将表格拆分为多个子表格。页面提供可视化预览，让用户确认拆分结果。

- **TaskCondition.vue**：条件设置页面，配置任务信息和单元格验证规则。用户可以设置任务名称、填报说明，并为每个单元格设置数据验证规则（如数据类型、格式、范围等），确保填报数据的准确性。

- **TaskRelease.vue**：任务发布页面，显示子表格列表和填报链接，跟踪完成情况。用户可以查看所有生成的子表格，获取每个子表格的填报链接，分发给相应的填报人员。页面实时显示各子表格的填报状态，方便用户跟踪进度。

- **TableFilling.vue**：表格填写页面，提供在线编辑功能和实时数据验证。填报人员通过链接访问此页面，按照设置的规则填写表格。系统实时验证数据格式，提供错误提示，确保数据质量。支持暂存填写进度和提交最终结果。

## 技术栈

- **Vue 3 (v3.5.24)**：采用Composition API，提供高效的组件开发体验
- **Vite (v7.2.4)**：快速的构建工具，提供热更新和优化的构建输出
- **Element Plus (v2.12.0)**：基于Vue 3的UI组件库，提供丰富的界面组件
- **Element Plus Icons (v2.3.2)**：Element Plus配套图标库
- **Pinia (v3.0.4)**：轻量级的状态管理库，管理应用的全局状态
- **Vue Router (v4.6.4)**：官方路由管理器，实现页面导航
- **Axios (v1.5.0)**：HTTP客户端，处理前后端数据交互
- **XLSX (v0.18.5)**：表格处理库，实现Excel文件的读取和导出
- **Handsontable (v16.2.0)**：强大的表格编辑组件，提供类似Excel的编辑体验
- **Spark MD5 (v3.0.2)**：文件哈希计算库，用于文件完整性验证

## 开发与运行

### 开发环境

**环境要求**
- Node.js 16+
- npm 或 yarn

**安装依赖**

```bash
cd frontend
npm install
```

**启动开发服务器**

```bash
npm run dev
```

开发服务器默认运行在 http://localhost:5173

### 构建生产版本

```bash
npm run build
```

构建产物将生成在 `frontend/dist` 目录下
